# soomin-project-board-final
자바 + 스프링부트 관련 기술로 게시판 만들기 - final 버전

# 개발환경
1. IDE: IntelliJ IDEA Community
2. Spring Boot 3.1.1
3. JDK 17
4. mysql
5. Spring Data JPA
6. Querydsl
7. Thymeleaf

# 게시판 주요기능 
1. 회원가입
2. 로그인/로그아웃
3. 게시글 생성, 조회, 수정, 삭제
4. 게시글 목록 조회 페이징처리
5. 댓글 생성, 조회, 수정, 삭제
